# Despesasapp
Programinha simples para despesas, feito com python 
O Setup estará responsavel por criar uma Pasta no seu "C:/Users/seu usuario/documents" Chamada DespesasApp, tbm criará outra pasta no %localappdata% do seu pc.
Caso o icone na tela não apareça crie o atalho com o arquivo no %localappdata%. 
Ao abrir o aplicativo e colocar suas despesas o DespesasApp criará um .txt com a soma das suas despesas e todas elas em "C:/Users/seu usuario/documents/DespesasAPP"
